# GraphQL API
---------------------------------------

* #### Development requirements
```bash
$ sudo apt install python3
$ sudo apt install python3-dev
$ sudo apt install python3-pip
```

* #### Create virtual env
```bash
$ sudo pip3 install pipenv 
```

* #### Setup environment
```bash
$ cd /var/www && git clone https://github.com/aram2726/graph.git
$ cd graph
$ pipenv install
$ pipenv run server
```
